How to play the game

This is a simple, fun game that allows the player to play a piano tune based on a random selection of piano notes generated during the game.

Input methods

The player can interact with the piano layout in two (2) ways - by clicking with a mouse or by using the keys on a computer keyboard.

Keyboard layout: The 12 notes on the piano correspond to the keys Z, S, X, D, C, V, G, B, H, N, J and M in the exact order of white and black piano keys. On a typical piano key notation, this would translate to C, Db, D, Eb, E, F, Gb, G, Ab, A, Bb and B. This makes it easy to recognize because this particular layout mirrors the layout of notes on a piano.

Scoring

The player get 1 point for every correct note played. However, the game ends when a wrong note is played.

Scores are ranked according the number of points accumulated. Amateur: 0-10 points. Intermediate: 11-20 points. Virtuoso: More than 20 points.
